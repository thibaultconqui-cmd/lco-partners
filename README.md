# LCO-Partners

Cabinet de chasse de têtes spécialisé IT & Conseil

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur http://localhost:3000

## Build

```bash
npm run build
```

## Structure

- `/src/app/page.tsx` - Page principale avec toutes les sections
- `/src/app/layout.tsx` - Layout principal
- `/src/app/globals.css` - Styles globaux avec TailwindCSS

## Fonctionnalités

- **Accueil** - Présentation du cabinet
- **Entreprises** - Section dédiée aux entreprises avec contenu expandable
- **Candidats** - Section dédiée aux candidats avec contenu expandable
- **Qui sommes-nous** - Présentation du fondateur et de l'approche
- **Contact** - Section de contact

## Technologies

- Next.js 14
- React 18
- TailwindCSS
- Lucide React (icônes)
- TypeScript
